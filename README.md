### 🌐 EchoSpot

> **共享设备高并发调度平台**

- 基于 **Spring Boot + Redis + RabbitMQ** 构建
- 实现设备发现、租借/归还逻辑及调度功能
- 解决**高并发下的资源竞争、状态一致性**以及**热点不均**问题
- 追求**高性能与高可用性**的设计方案

👉 [https://github.com/zeroverload/EchoSpot](https://github.com/zeroverload/EchoSpot)

---

### 🧠 Local Mind

> **具备记忆与 RAG 能力的本地知识库**

- 基于 **Streamlit + Ollama + LanceDB** 开发
- 支持笔记记录、PDF 文档导入及语义搜索
- 结合 **短期+长期记忆 + 混合检索** 技术
- 实现**完全本地化部署**的上下文感知问答系统

👉 [https://github.com/zeroverload/Local_mind](https://github.com/zeroverload/Local_mind)

### 🔖 Bookmark Home

> **本地优先的 Edge 浏览器扩展，打造极简收藏夹主页**

- 将新标签页转换为**收藏夹卡片展示页**
- 支持 **搜索、别名、用途标签** 以及 **卡片排序**
- 提供编辑模式，支持在不同收藏夹间**快速移动书签**
- 遵循**本地优先**原则，轻量且高效

👉 [https://github.com/zeroverload/edge-bookmark-home](https://github.com/zeroverload/edge-bookmark-home)
